# 🚀 AiTool 桌面工作助手 — 彻底终结日常繁琐、反复操作与频繁寻档的极客效率中转站

[![Platform](https://img.shields.io/badge/platform-Windows-orange.svg)]()
[![Python](https://img.shields.io/badge/Python-3.10%2B-blue)]()

在日常工作中，你是否也深陷于以下痛点？
* **😫 反复寻找文件/文件夹**：手头有几个高频深度使用的配置文件、素材图或临时文档，每次想找时都要在十几层深的目录里层层点击，或者在密密麻麻的桌面堆里瞎找，效率极低。
* **🔁 机械而烦人的日常搬运**：比如每天都要把“打包生成的资源”手动覆盖拷贝到“SVN测试服路径”，还要分流到“备用路径”，这种**重复的手动复制粘贴**每天要做好几次，极其枯燥。
* **🛠️ 频繁调用不同工具/脚本**：一会儿要跑个 Python 清理脚本，一会儿要启动个 Bat 编译脚本，一会儿又要打开特定的内网管理网页，整个屏幕堆满了各种黑窗口和浏览器标签页，乱成一锅粥。

`AiTool 桌面工作助手` 就是为了**彻底斩断这些无意义的机械重复与心智消耗**而生的。
它是一台专门钉在屏幕边缘、永不碍眼的**拖拽式资产中转站与多路效率执行器**。

---

## 🎯 它是如何解决你的痛点的？

### 1. 终结“频繁寻档”：随手拖入，随时调遣
* **拖入暂存**：不论是高深路径里的 `.json`、美术美术打包的 `.png`，还是测试发来的 `.zip`，随手往软件界面里一拖，它就会立刻安稳躺在“中转站”中。
* **双击即开 / 悬停透视**：双击直接调用系统关联程序打开该文件。鼠标往上一放，下方状态栏立刻以暗灰低调字体显示**完整物理路径**。
* **右键一键定位**：对中转站文件点击**鼠标右键**，会自动唤起 Windows 资源管理器**直接跳转到该文件所在的文件夹，并自动高亮选中它**！再也不用去点那长长的地址栏。
- **拖出复制**：需要把暂存文件发给同事？长按它，直接从软件界面里拖拽丢进微信、钉钉或任何其他文件夹，即刻完成复制！

### 2. 终结“机械搬运”：配置一次，一键双通道同步
* **多路径自动覆盖复制**：支持配置**双通道路径覆盖**（最多 4 个源-目的对应路径）。只需一次配置，当你双击卡片或中转站一键同步时，它就会在后台**静默地并行复制多路数据**，并为你弹出一个非阻塞的、带有动态旋转进度条和进度标签的现代 HUD 悬浮窗，不卡死、有回馈。

### 3. 终结“脚本繁杂”：拖进来就是快捷指令
* 当你将任何一个 `.bat / .cmd / .py` 脚本文件，或者 `.lnk / .exe` 快捷方式，甚至是某个**网页链接**拖入界面时，`AiTool 桌面工作助手` 就会聪明地认出它：
  - 拖入脚本 ➔ 自动为您创建 “执行该脚本” 的快捷卡片。
  - 拖入网页 ➔ 自动创建 “一键唤起浏览器打开该网页” 的快捷卡片。
  - 拖入软件 ➔ 自动创建 “一键启动应用” 的快捷卡片。
* 所有这些快捷指令都会自动收纳在“快捷动作”卡片中，并且右侧内置精美 `>` 按钮，可随时自定义、删除和编辑参数。

### 4. 彻底的绿色和不打扰
* **Alt + A 闪现退避**：工作时按下 `Alt + A` 瞬间在鼠标位置唤起，用完再按 `Alt + A` 瞬间隐退，或者在设置里开启“始终置顶”，成为你屏幕右侧永远待命的侧边栏。
- **系统托盘**：关闭时自动缩入 Windows 系统托盘（亮起一颗代表极致效率的科技感微光火箭图标），不占用你的任何任务栏空间，并支持可选开机自启动。

## ✨ 核心功能全景

- **📂 全局万物皆可拖放 (`Unified Drag-and-Drop`)**
  - 支持直接拖拽链接、文字、本地文件，按后缀自动智能分类，无需任何繁琐的添加流程。

- **📦 极客文件资产中转站 (`Asset Staging Station`)**
  - **右键快速定位**：鼠标右键直接打开所在文件夹并定位/选中文件。
  - **悬停路径显示**：底部状态栏低调呈现文件的完整绝对路径，有效防混淆。

- **⚡ 多通道高频动作（快捷卡片）**
  - **SVN 自动提交 (SVN Commit)**：一键唤起指定路径的 SVN 提交面板。
  - **动作快速自定义**：可一键动态增删改快捷指令卡片的标题、描述与执行参数。


## 🚀 快速开始

### 方式一：源码运行

1. 确保安装了 Python 3.10+ 环境。
2. 安装依赖项：
   ```bash
   pip install customtkinter tkinterdnd2 pystray Pillow keyboard
   ```
3. 运行程序：
   ```bash
   python run_desktop_tool.py
   ```

### 方式二：编译打包 (PyInstaller + UPX)

如果您想将其编译打包为精简、绿色的单文件 `AiTool.exe`，我们已经完成了极致的体积优化过滤（排除 numpy, scipy 等大体积包），您可以直接运行：

```bash
pyinstaller --clean --noconfirm --onefile --windowed --add-data "src/aitool_desktop;src/aitool_desktop" --exclude-module numpy --exclude-module matplotlib --exclude-module scipy --exclude-module xml --exclude-module multiprocessing --name "AiTool" run_desktop_tool.py
```
